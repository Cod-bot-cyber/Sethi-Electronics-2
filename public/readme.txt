This is the public static assets directory.
Any files placed here (such as .png, .jpg, .svg, or .gif) will be served directly at the root path of the application.

For example, if you save an image named 'headphones.png' inside this folder, you can access and reference it in your browser or application code using:
- Link / Relative Path: /headphones.png
- HTML / JSX: <img src="/headphones.png" />
